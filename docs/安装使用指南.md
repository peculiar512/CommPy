# CommPy 安装使用指南

## 一、环境要求
- Python 版本：3.8 ~ 3.12（推荐3.9/3.10，兼容性最好）
- 操作系统：Windows 10/11、macOS、Linux
- 核心依赖：NumPy >= 1.20, SciPy >= 1.7, Matplotlib >= 3.5

## 二、Windows 详细安装步骤
### 步骤1：安装Python
1. 打开 Python 官网，下载 3.9 或 3.10 版本的安装包
2. 运行安装包，**务必勾选底部的「Add Python to PATH」**，然后点击「Install Now」
3. 安装完成后，按 `Win+R` 输入 `cmd` 回车，执行 `python --version`，显示版本号即为安装成功。

### 步骤2：获取项目代码
1.Git克隆
```bash
git clone https://github.com/peculiar512/CommPy.git
cd CommPy
2.执行以下命令，输出版本号即为安装成功：
pip uninstall numpy -y
pip install numpy==1.24.4
三、常见报错与解决方法
1. NumPy版本不兼容
报错现象：运行时提示 module 'numpy' has no attribute 'xxx' 或版本冲突
解决方法：卸载当前版本，安装兼容版本
pip uninstall numpy -y
pip install numpy==1.24.4
2. 提示「ModuleNotFoundError: No module named 'xxx'」
原因：缺少对应的依赖库
解决方法：安装缺失的库，比如缺matplotlib就执行：
pip install matplotlib
3. pip命令找不到
原因：Python没有加入系统环境变量
解决方法：重新运行Python安装包，选择「Modify」，勾选「Add Python to environment variables」后完成安装。
四、功能调用示例
示例1：BPSK误码率仿真
import numpy as np
from commpy.modulation import bpsk_modulate, bpsk_demodulate
from commpy.channel import awgn_channel

# 生成随机比特
bits = np.random.randint(0, 2, 10000)
# BPSK调制
mod_signal = bpsk_modulate(bits)
# 通过AWGN信道，SNR=10dB
rx_signal = awgn_channel(mod_signal, snr=10)
# 解调
rx_bits = bpsk_demodulate(rx_signal)
# 计算误码率
ber = np.sum(bits != rx_bits) / len(bits)
print(f"SNR=10dB时，误码率：{ber}")
示例二：汉明码编译码
from commpy.channel_coding import hamming_encode, hamming_decode

info_bits = [1, 0, 1, 1]
# 编码
code_bits = hamming_encode(info_bits)
print(f"编码后：{code_bits}")
# 译码
decoded_bits = hamming_decode(code_bits)
print(f"译码后：{decoded_bits}")

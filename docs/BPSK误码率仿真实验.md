# 实验1：BPSK调制解调与误码率仿真实验

## 一、实验目的
1. 掌握BPSK调制与相干解调的基本原理
2. 理解误码率与信噪比的关系
3. 通过Python仿真验证BPSK的理论误码率公式

## 二、实验原理
### 2.1 BPSK调制解调原理
BPSK用载波的两种相位表示二进制信息，表达式为：
s_1(t) = A cos(2πf_c t),  发送"1"
s_0(t) = A cos(2πf_c t + π),  发送"0"

接收端采用相干解调，将接收信号与本地载波相乘，经低通滤波后进行门限判决，恢复原始比特。

### 2.2 误码率公式
AWGN信道下，BPSK的理论误码率为：
P_e = Q(√(2E_b/N_0))
其中Q函数为互补误差函数，E_b为比特能量，N_0为噪声功率谱密度。

## 三、实验环境
- 操作系统：Windows 10/11
- Python版本：3.9
- 依赖库：NumPy 1.24, Matplotlib 3.7

## 四、实验步骤
1. 生成随机二进制比特序列（长度100000）
2. 对比特序列进行BPSK调制
3. 信号通过不同信噪比的AWGN信道
4. 对接收信号进行相干解调与判决
5. 统计误码数，计算实际误码率
6. 绘制误码率曲线，与理论值对比

### 5.2 结果分析
- 随着信噪比SNR增大，误码率逐渐降低
- 仿真得到的误码率与理论公式计算结果基本吻合
- 低信噪比下仿真误差略大，原因是比特序列长度有限，误码数统计样本不足

## 六、实验结论
1. BPSK调制解调系统在AWGN信道下的误码率随信噪比提升呈指数下降
2. 仿真结果与理论推导一致，验证了BPSK原理的正确性
3. 增加仿真比特长度可以提升低信噪比下的结果精度

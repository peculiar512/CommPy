# 信道模型原理

## 一、概述
信道模型用于模拟信号在实际传输过程中受到的损伤，是通信仿真的基础模块。
本项目实现的信道模型：AWGN高斯白噪声信道、多径瑞利衰落信道等。

## 二、核心信道模型
### 2.1 AWGN加性高斯白噪声信道
- **原理**：在发送信号上叠加服从高斯分布的白噪声，模拟热噪声等随机干扰。
- 噪声功率与信噪比关系：`SNR = 10lg(Ps/Pn)`，通过调整噪声功率控制信道质量。
- 适用场景：模拟有线信道、视距无线信道的基础噪声环境。

### 2.2 瑞利衰落信道
- **原理**：模拟多径传播效应，信号经多条路径到达接收端，幅度服从瑞利分布，相位均匀分布。
- 产生方法：通过两个独立高斯随机变量的平方和开根号得到瑞利分布幅度。
- 适用场景：模拟非视距、多径丰富的无线移动通信场景。

## 三、项目实现逻辑
- 代码位置：`commpy/channel/` 目录
- 核心函数：
  - `awgn_channel()`：添加高斯白噪声
  - `rayleigh_channel()`：瑞利衰落信道
- 输入参数：输入信号、信噪比SNR、多径数量

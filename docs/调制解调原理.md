# 调制解调算法原理

## 一、概述
数字调制的作用是将二进制数字基带信号，变换为适合在信道中传输的带通信号；解调则是接收端从带通信号中恢复出原始数字信息。
本项目实现的调制方式：BPSK、QPSK、16QAM、64QAM等。

## 二、典型调制原理
### 2.1 BPSK（二进制相移键控）
- **基本原理**：用载波的0°和180°两种相位分别表示二进制0和1。
- 表达式：`s(t) = A cos(2πf_c t + φ)`，φ=0对应1，φ=π对应0。
- **解调方式**：相干解调，本地载波与接收信号相乘，经低通滤波后判决恢复数据。

### 2.2 QAM（正交幅度调制）
- **基本原理**：同时利用载波的幅度和相位携带信息，频谱利用率高。
- 星座图：I路（同相）和Q路（正交）分别承载信息，合成后形成矩形星座点。
- 解调：分别对I、Q两路进行采样判决，合并得到原始比特序列。

## 三、项目实现逻辑
- 代码位置：`commpy/modulation/` 目录
- 核心函数：
  - `bpsk_modulate()`：BPSK调制
  - `qam_demodulate()`：QAM解调
- 关键参数：调制阶数、载波频率、采样率

## 四、性能对比
- 相同信噪比下，调制阶数越高，频谱利用率越高，误码率越高。
- BPSK抗噪声性能最优，适合低信噪比场景；高阶QAM适合高速率通信场景。
